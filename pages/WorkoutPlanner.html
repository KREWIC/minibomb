Workout Planner

Objective

This program is designed to create a random workout based on a set of muscle groups you input. The program is pretty barebones for now, but will improve probably maybe at some point in the future. The main purpose of this program is to use python to pull and utilize data from a seperate script.

The Code

This program uses a few different concepts. I've encorporated the random library so I can pull from the different arrays I have set up in my dataset. The main program pulls from the dataset and will randomly shuffle them. The program will then pull from those shuffled list after taking an input from the user on the muscle groups they want to work that day.
